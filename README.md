# Analog-Amplifier-Design-
Design an amplifier with the following specifications. 
Load resistance, RL = 1k ohm. One of the terminals of RL must be grounded, and the other 
terminal must be AC-coupled with the rest of the network.  
HD1 / HD3 = 40 dB for output pk-pk swing of 1V when a sinusoidal input of frequency 100 
MHz is applied. Use of an inductor is not allowed.  The maximum size of a capacitor is 100pF. 
There is no specification on the input swing, and the number and type of transistors that you 
can use. 
A single supply VDD = 1.8V is available for you, and the input sinusoidal signal is centred 
around zero DC.
